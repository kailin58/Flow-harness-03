## 配置文件

### 配置位置

| 文件 | 用途 | 优先级 |
|------|------|--------|
| `~/.config/rtk/config.toml` | 全局配置 | 低 |
| `.rtk/config.toml` | 项目配置 | 高 |
| `.rtk/filters/*.toml` | 项目过滤器 | 最高 |

### 配置示例

```toml
# ~/.config/rtk/config.toml

[tracking]
enabled = true
database_path = "~/.local/share/rtk/tracking.db"
retention_days = 90

[filters]
max_lines = 100
show_warnings = false
strip_ansi = true

[hooks]
auto_install = true
supported_tools = ["claude-code", "cursor", "copilot"]
```
