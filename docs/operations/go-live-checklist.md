# Go-Live Checklist

## 发布前

- [ ] CI 全绿（lint、test、依赖扫描）。
- [ ] 核心 SQL 已提供 `EXPLAIN ANALYZE` 结果并归档。
- [ ] OpenAPI 已冻结鉴权、限流、超时、标准错误码。
- [ ] 监控看板与告警规则已发布到生产环境。
- [ ] Runbook（超时、积压、重放、回滚）已演练。

## 发布中

- [ ] 使用灰度发布（5% -> 20% -> 50% -> 100%）。
- [ ] 实时观察 P95/P99、错误率、429/5xx 比例。
- [ ] 任一核心 SLO 失守时自动暂停扩大流量。

## 发布后

- [ ] 核心链路对账通过（订单、积分、佣金）。
- [ ] 24 小时内无 P1 级告警。
- [ ] 输出发布复盘与容量修正建议。
