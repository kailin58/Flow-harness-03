# 用户入口商品详情页底部主操作区约束记录（2026-04-09）

## 背景

在明确购物车页底部结算栏与结算页底部提交订单栏都采用“页面内常驻显示”的口径后，商品详情页也需要统一为同类设计，避免交易主链路三段在核心操作区表达上不一致。

## 本次结论

- 商品详情页底部主操作区在用户停留该页时应常驻显示
- 商品详情内容区负责滚动，底部主操作区不跟随内容区滚动离开视口
- 底部主操作区稳定承载 `加入购物车` 与 `立即购买` 两个主动作
- 商品详情内容区需为底部主操作区预留安全高度，避免最后一段决策信息被遮挡

- 交易主链路中的商品详情页、购物车页、结算页现已统一采用“底部常驻主操作区”口径
