# 外部渠道平台范围口径同步记录（2026-04-09）

## 背景

在补充 `meeting/5-1用户统一入口.md` 与 `meeting/5-2平台后台入口.md` 时，发现 `meeting/` 中关于第三方渠道平台范围的表述不完全一致，部分文档写为 `淘宝/抖音/小红书/百度/快手`，部分位置已开始补充 `京东`、`唯品会`。

## 本次统一口径

- 当前第三方渠道平台范围统一表述为：如 `淘宝`、`抖音`、`小红书`、`京东`、`唯品会`、`快手`、`百度` 等平台。
- `用户统一入口` 负责承接这些平台的导流、回站与来源参数透传，不把第三方渠道流量单独拆成新的前台入口。
- `外部渠道与店铺管理端` 负责这些平台的接入、授权、能力矩阵、映射关系、店铺状态治理。
- 平台范围口径不等于一期全部同时接入；一期试点平台数量与顺序单独在范围冻结与优先级文档中约定。
